# 📊 Painel Macroeconômico Brasileiro 2025–2026

Dashboard interativo desenvolvido para centralizar os principais indicadores econômicos do Brasil em uma única aplicação visual, leve e acessível.

O projeto reúne informações macroeconômicas como inflação, taxa Selic, PIB, desemprego, câmbio e indicadores de renda em um painel responsivo construído com HTML, CSS e JavaScript puro.

---

# 🚀 Objetivo do Projeto

A ideia surgiu da necessidade de acompanhar indicadores econômicos sem precisar abrir diversos sites diferentes.

O dashboard foi desenvolvido para:

* centralizar dados econômicos relevantes
* facilitar análises comparativas
* transformar indicadores complexos em visualizações acessíveis
* criar uma experiência rápida e portátil
* servir como projeto de portfólio na área de dados/front-end analítico

# 🧠 Insight Econômico

Durante o desenvolvimento do projeto, uma análise chamou atenção:

Embora o IPCA oficial tenha fechado próximo de 4%, o grupo de alimentação apresentou alta superior a 10%.

Isso mostra como a percepção real da inflação pode ser diferente do índice agregado divulgado oficialmente, principalmente para famílias que concentram grande parte da renda em alimentação.

O dashboard foi pensado justamente para permitir esse tipo de análise visual e contextual.
